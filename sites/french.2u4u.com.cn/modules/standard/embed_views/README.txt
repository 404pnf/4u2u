-- SUMMARY --

This is just a simple module that creates an embed views display specifically 
for embedding views on your site. This module does not add *ANY* additional 
features to views it replicates the same functionality of the 'defaults' views 
display.

-- REQUIREMENTS --

Views 6.x-2.x

-- INSTALLATION --

Install as usual, see http://drupal.org/node/70151 for further information.
