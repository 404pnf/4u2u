ABOUT:
   The dash media player is an open source GPL - license free - full featured Flash media player written in ActionScript 3.
   It was designed to dynamically play any media thrown it's way, whether it be Video or Audio, etc.

   Although this media player was designed to be used within a Content Management environment, such as Drupal, you can
   use this player for other applications outside of a CMS.
